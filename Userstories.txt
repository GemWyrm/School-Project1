As a player, I want to quickly see what my character can do so that I can focus on the play instead of looking things up.
As a Game Master, I want to be able to see all my players' sheets so that I can quickly know their capabilities.
As a player, I want to be able to quickly input my character's homebrew and RAW (Rules As Written) abilities so that I don't have to spend a lot of time creating the character sheet.
